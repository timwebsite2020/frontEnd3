ISPO (日本語）

インドネシアの持続可能なアブラヤシ農園業の認証システムに関する

インドネシア共和国の大統領令

2020年44号


全能の神様の恵みにより、

インドネシア共和国の大統領は、


勘案事項：

  1. インドネシアのアブラヤシ農園業は大規模な労働力を吸収し、国の外国為替に貢献するため、アブラヤシ農園のための効果的、効率的、公平かつ持続可能な管理システムが国の経済発展を支えるために必要です；

  2. 法令に従ってアブラヤシ農園業が社会的、経済的、環境的に適切であることをより確実にするには、インドネシアの持続可能なアブラヤシ農園業の認証システムの実施を改善する必要があります；

  3. インドネシアの持続可能なアブラヤシ農園業の認証システムを管理する法律と規制は、国際的な進展と法的ニーズに沿っていないため、大統領規制で取り替えて規制する必要があります。

  4. a、b、cに述べた点を考慮したうえ、インドネシアの持続可能なアブラヤシ農園業の認証システムに関するインドネシア共和国の大統領令を定める必要がある。

参照事項：

インドネシア共和国1945年憲法第4５条 ( 1 ) 項；

以下を決定した：

インドネシアの持続可能なアブラヤシ農園業の認証システムに関する

インドネシア共和国の大統領令を定める。

第I章

一般規定

第1条

この大統領令で用いる用語の定義は以下のとおりとする：

  1. アブラヤシ農園業とは、天然資源、人的資源、生産設備、道具と機械、栽培、収穫、加工、アブラヤシのマーケティングを管理するすべての活動を指す。

  2. アブラヤシ農園事業とは、アブラヤシ農園からの商品やサービスを生産する事業を指す。

  3. インドネシアの持続可能なアブラヤシ農園業またはIndonesian Sustainable Palm Oil（以下、「ISPO」と称する）とは、法的規定に基づき、経済的に実現可能で、社会文化的で、環境にやさしいアブラヤシ農園業システムを指す。

  4. ISPO認証は、アブラヤシ農園業の製品またはガバナンスがISPOの原則と基準を満たしていることの書面による保証の提供に関連するアブラヤシ農園事業に対する一連の適合性評価活動を指す。

  5. 農園事業主体（以下）、「事業主体」と称する）とはアブラヤシ農園事業を管理するアブラヤシ農民またはアブラヤシ農園企業を指す。

  6. アブラヤシ農民（以下、「農民」と称する）とは、一定の規模に達していない事業規模でアブラヤシ農園事業を実施するインドネシアの個人を指す。

  7. アブラヤシ農園企業（以下「農園企業」と称する）とは、インドネシアの法律に従って設立され、一定の規模でアブラヤシ農園事業を運営するインドネシアの領土に本社を置く法人を指す。

  8. アブラヤシ農園業の結果とは、すべてのアブラヤシ農園作物の製品と、主要製品、保存期間を延長するための加工製品および副産物からなるそれらの加工品を指す。

  9. 国家認定委員会（以下、「KAN」と称する）とは、適合性評価機関の認定の分野で義務と責任を負う非構造的機関を指す。

  10. ISPO認証機関とは、ISPO認証を実行し、ISPO認証を発行する独立した適合性評価機関を指す。

  11. 中央政府とは、インドネシア共和国1945年憲法に述べられたように、統一インドネシア共和国の政権を掌握するインドネシア共和国大統領を指す。

  12. 地方政府とは、地方政府を組織する州知事、県知事または市長、および地方要員を指す

  13. 大臣とは、農園業部門の政府業務を主催する大臣を指す。


第2条

この大統領令のの適用範囲は以下のとおり：

  1. ISPO認証；

  2. 機関；

  3. 受け入れ、市場競争力、および参加；

  4. 指導と監督；および

  5. ペナルティ。


第3条

ISPO認証システムの実装は、以下を目的とする：

  1. ISPOの原則と基準に従って、アブラヤシ農園業の管理と開発を確実にし、向上する；

  2. 国内および国際市場でのアブラヤシ農園業の結果の受け入れと競争力を高める；および

  3. 温室効果ガスの排出削減を加速する取り組みを強化する。


第II章

インドネシアの持続可能なアブラヤシ農園業の認証


第1部

一般


第4条

  1. インドネシアの持続可能なアブラヤシ農園業を確保するために、ISPO認証が行われます。

  2. 第1項に述べたISPO認証は、以下の原則を適用することによって実施されるものとします。

  1. 法令の遵守；

  2. 良好農園業慣行の適用；

  3. 環境、天然資源、生物多様性の管理；

  4. 労働責任；

  5. 社会的責任とコミュニティの経済的エンパワーメント；

  6. 透明性の適用；および

  7. 継続的な事業の改善。

  1. 第2項に述べた原則は、ISPO基準に記載されています。

  2. 第2項および第3項に述べたISPOの原則および基準に関する更なる規定は、大臣令によって規定する。


第5条

  1. アブラヤシ農園事業の場合、ISPO認証は、第4条に記載されているように実行する必要があります。

  2. 第1項に述べたアブラヤシ農園事業は、次のもので構成されています。

  1. アブラヤシ農園作物栽培業；

  2. アブラヤシ農園作物加工業；および

  3. アブラヤシ農園作物栽培業およびアブラヤシ農園作物加工業のの統合。

  1. 第1項に述べたISPO認証は、以下の事業主体によって提出されるものとします。

  1. 農園企業；または

  2. 農民。

  1. 第3項の ( b ) に述べたように農民によって提出されたISPO認証は、個別にまたはグループで実行できます。

  2. 第4項 に述べたグループは、小規模農民グループ、結合された小規模農民グループ、または協同組合の形をとることがあります。


第6条

  1. 第5条に述べたISPO認証義務の規定に違反する事業主体は、大臣による行政処分の対象となる：

  2. 第1項に述べた行政処分は以下の通りである：

  1. 書面による警告；

  2. 罰金；

  3. アブラヤシ農園事業活動の一時停止；

  4. ISPO証書の一時停止；または

  5. ISPO証書の撤回。

  1. 第1項および第2項に述べた行政処分の適用に関する更なる規定は、大臣令によって規定する。


第2部

ISPO認証機関


第7条

  1. 第5条に述べたISPO認証は、ISPO認証機関によって実行されます。

  2. 第1項に述べたISPO認証機関は、

  1. 標準化および適合性評価の分野における法的規定に従って、KANによって既に認定された；および

  2. 農園業分野での政府関連業務を担当する省庁に既に登録された。

  1. 第1項に述べたISPO認証機関は、次の義務があります。

  1. 事業主体に対してISPOの原則と基準への準拠の適合性の評価を実施します；

  2. ISPO認証活動の結果に基づいて、アブラヤシ農園事業のISPO認証を発行、一時停止または撤回する；

  3. ISPO認証を既に取得しているアブラヤシ農園事業の年次評価を実施する；および

  4. ISPO認証の実施に関する苦情と異議申し立てのフォローアップする。


第3部

認証の要件と手順


第8条

  1. 第5条3項の ( a ) に述べた農園企業は、ISPO認証の申請書をISPO認定機関に提出し、第4条に述べたISPO原則と基準の履行を評価します。

  2. 第1項 に述べた申請書には、次の文書が同封されています。

  1. 農園事業許可；

  2. 土地証書；

  3. 環境許可；

  4. 農園事業許可の授与者からの農園事業評価の決定。

  1. 第5条3甲の ( b ) に述べた農民は、次の文書を添付してISPO認証機関にISPO認証の申請書を提出します。

  1. 農園事業登録証書；

  2. 土地証書。


第9条

  1. ISPO認証の申請は、事業主体によってISPO認定機関に提出されます。

  2. ISPO認証機関は、第8条に述べられているように、申請要件文書の完全性を確認します。

  3. この申請が第8条にのべた要件を満たしている場合、ISPO認証機関はISPO認定を実施します。

  4. この申請が第8条にのべた要件を満たしない場合、ISPO認証機関はISPO認証の申請を拒否します。

  5. ISPO認証機関は、第4項に述べた申請の拒否およびその理由に関する通知を事業主体に配信します。

ISPO認定機関は、第4条に記載されているISPOの原則と基準の履行を評価することにより、ISPO認定を実施します。


第11条

  1. 事業主体がISPOの原則と基準を満たしている場合、ISPO認定機関はISPO認証を発行します。

  2. 事業主体がISPOの原則と基準を満たしていないと見なされた場合、ISPO認証機関は改善を行うし、要件を完了するように事業主体に推奨を提出します。

  3. 事業主体が改善を行った、またはISPOの原則と基準を満たすための要件を完了した場合、ISPO認証機関はISPO認証を発行します。

  4. 事業主体が改善を行わなかったり、ISPOの原則や基準を満たすための要件を満たさなかったりした場合、ISPO認定プロセスは続行されず、ISPO認定の申請はキャンセルされます。


第12条

  1. ISPO認証機関は、以下の事項についてISPO委員会に報告を提出する必要があります。

  1. 発行されたISPO認証；および

  2. ISPOの原則と基準を満たすために改善を行ったり要件を完了したりしている事業主体。

  1. 第1項に述べられているような報告を提出しないISPO認証機関は、大臣による行政処分の対象となる。

  2. 第1項に述べた行政処分は、以下の通りです。

  1. 書面による警告 ；

  2. ISPO認証機関として農園業分野での政府関連業務を担当する省庁のリストから削除されました。

  1. 第2項および第3項に述べた行政処分の適用に関する更なる規定は、大臣令によって規定する。

  2. 第3項に述べた行政処分に加えて、ISPO認証機関は、標準化および適合性評価の分野の法的規定に従って、ISPO認証機関としての認定証明書の一時停止または取り消しの形で、KANによる制裁の対象となる場合があります。


第13条

  1. ISPO認証は5年間以内に有効とする。

  2. 第1項に述べたISPO認証の期間が終了する前に、事業主体はISPO認証を再び提出しなければなりません。


第14条

ISPO認定手続きに関する更なる規定は、大臣令によって規定する。


第4部

監視


第15条

  1. ISPOによって認証されたアブラヤシ農園事業は、ISPO認証機関による監視を受ける必要があります。

  2. 第1項に述べた監視は、事業主体によるISPOの原則と基準の履行を確認するために実行されます。

  3. 監視に関する更なる規定は、大臣令によって規定する。


第16条

  1. 第4条に述べたISPOの原則と基準を満たさないアブラヤシ農園事業は、大臣による行政処分の対象となる。

  2. 第1項に述べた行政処分は、以下の通りです。

  1. ISPO認証の一時停止；または

  2. ISPO認証の撤回。

  1. 第2項に述べた行政処分の適用に関する更なる規定は、大臣令によって規定する。


第17条

  1. 第15条第1項に述べた監視を行わないISPO認証機関は、大臣による行政処分の対象となる。

  2. 第1項に述べた行政処分は、以下のA通りです。

  1. 書面による警告 ；

  2. ISPO認証機関として農園業分野での政府関連業務を担当する省庁のリストから削除されました。

  1. 第2項に述べた行政処分の適用に関する更なる規定は、大臣令によって規定する。

  2. 第2項に述べた行政処分に加えて、ISPO認証機関は、標準化および適合性評価の分野の法的規定に従って、ISPO認証機関としての認定証明書の一時停止または取り消しの形で、KANによる制裁の対象となる場合があります。


第5部

資金


第18条

  1. 第5条3項の ( a ) に述べた農園業企業が提出したISPO認証の資金は、各農園業企業が負担するものとします。

  2. 農民から提出されたISPO認定の資金は、以下から調達できます。

  1. 国家予算；

  2. 自治体予算；および/または

  3. その他の正当な資金源。

  1. 第2項に述べた資金調達は、農民、農民グループ、または協同組合を通じて行われ、最初のISPO認定期間中に提供されます。

  2. 第2項に述べたISPO認証のコストおよび第3項に述べた施設に関する更なる規定は、大臣令によって規定する。


第III章

機関


第1部

ISPO 委員会


第19条

  1. ISPOの実践と管理を調整するコンテキストで、ISPO委員会が設立されました。

  2. 第1項に述べたISPO委員会は、次の役割を担っています。

  1. ISPO運営委員会によって運用ポリシーに確立されたISPOの管理に関する一般的なポリシーを説明する；

  2. ISPOの原則と基準を策定する；

  3. ISPOの原則と基準への履行の各レベルの評価基準を策定します；

  4. ISPO認証の要件とスキームを策定する；

  5. 良好な農園業ガバナンスを維持するために、ISPO認証システムの実装を評価します。

  6. ISPO認証情報システムを構築する；および

  7. 省庁、機関、地方自治体、およびISPO認定の管理と実施に関連して必要と思われる他の関係者と調整する。

  1. ISPO委員会は、第2項 ( f )に述べた ISPO認証情報システムを構築および開発し、事業主体がISPO認証を簡単に取得できるように統合および電子的に統合されたデータ共有システムを実装します。


第20条

  1. 第19条に述べたISPO委員会は大臣が議長を務め、政府の要素、企業団体、学者、独立したオブザーバーで構成されています。

  2. 第1項に述べられているように、政府に由来するISPO委員会のメンバーシップ要素は職権です。

  3. 第1項に述べた独立オブザーバーに由来するISPO委員会のメンバーシップ要素は、インドネシアの法人またはインドネシアの市民である非政府組織であり、社会、経済、および環境の分野で懸念のある農園業を担当しています。


第2部

ISPO運営委員会

  1. ISPO委員会の任務を遂行するための方針の方向性を提供するために、ISPO運営委員会が設立されました。

  2. 第1項に述べたISPO運営委員会は、以下の義務を負っています。

  1. ISPOシステムおよびメカニズムにおける一般的なポリシーを決定する；

  2. 上記のa点に述べたポリシーを監督および評価する。および

  3. ISPO委員会の会員構成を決定します。

  1. 第1項に述べたISPO運営委員会は、以下の通りです。

  1. 会長 ：経済分野での政府関連業務を担当する大臣

  2. 日々会長 ：大臣

  3. 会員 ：1. 環境・林業分野での政府関連業務を担当する大臣；

2. 農業および空間計画の分野での政府関連業務を担当する大臣；

3. 商業分野での政府関連業務を担当する大臣；

4. 工業分野での政府関連業務を担当する大臣；

5. 国内分野での政府関連業務を担当する大臣；および

6. 国家標準化分野での政府関連業務を担当する大臣；


第22条

ISPO運営委員会の組織と作業手順に関する更なる規定は、ISPO運営委員会の議長として経済分野での政府関連業務を担当する大臣によって規定する。


第IV章

受け入れ、市場競争力、参加


第23条

国内および国際的に市場の受け入れと競争力を高めるために、中央政府と地方自政府は以下の取り組みを行っています。

  1. 撒布；

  2. 擁護；

  3. 国際外交；および

  4. 製品の承認と適合性評価システムの認識。


第24条

  1. 社会、事業主体、および利害関係者は、ISPO認定の管理と実装に参加できます。

  2. 第1項に述べた参加は、以下のように行うことができます。

  1. ISPO認証の管理と実践に関する提案を提案する；

  2. ISPO認証の管理と実践に関する情報を要求し、取得する；

  3. ISPO認証の管理と実践からの誤用または逸脱を政府、ISPO委員会、KAN、および/またはISPO認証機関に報告する；および/または

  4. 国内および国際市場におけるISPOの受け入れと競争力およびとインドネシアのアブラヤシ農園業の結果のを高めるために、政府と協力する。


第V章

指導と監督


第25条

  1. 中央政府と地方政府は、事業主体のためのISPO認定の実施に関する指導と監督を提供します。

  2. 第1項に述べた中央政府による指導および監督は、法律の規定に基づいて、大臣および機関の長がその職務、機能および権限に従って実施するものとする。

  3. 第1項に述べた地方政府による指導および監督は、法律の規定に基づいて、知事がその職務、機能および権限に従って実施するものとする。

  4. 第1項に述べた農民への指導には、ISPOの原則と基準の準備と履行が含まれます。


第VI章

経過規定


第26条

  1. この大統領令が発効する前に発行されたISPO認証は、次の条件によりISPO認証の有効期限が切れるまで有効であると宣言されます。

  1. ISPO認定機関による監視の結果に基づいて実行されるこの大統領令で規制されているISPOの原則と基準に基づいて、事業主体はISPOの適用に調整を行います。

  2. ISPO認証機関は、上記の ( a )に述べた監視に基づいて、指定された期間に従って監視レポートを発行します。

  3. 事業主体は、上記の ( a )に述べた監視レポートに基づいて、指定された期間に従ってISPOの適用を調整する必要があります。

  4. 上記の ( b )に述べた監視レポートに基き、

  1. 事業主体がISPOの原則と基準を適用した場合、ISPO証明書は引き続き有効です。

  2. 事業主体がISPOの原則と基準を適用していない場合、事業主体は次の監視まで調整を行うように要求されます。

  1. 事業主体が上記の ( b ) の2点に述べた調整を行わなかった場合、この大統領令が施行される前に発行されたISPO認証は取り消されます。

  1. 第1項に述べたISPO認証の調整に関する更なる規定は、ISPO委員会の会長として大臣によって規定する。


第VII章

最終規定


第27条

この大統領令が発効した場合、

  1. 第5条第3項の ( a ) に述べた農園企業用のISPO認証は、この大統領規則が法律化された日付から有効とする。

  2. 第5条第3項の ( b ) に述べた農民のためのISPO認証は、この大統領令が法律化されてから5年間以内に有効とする。


第28条

この大統領令が施行されると、ISPO認証を管理するすべての法律および規則は、この大統領令の規定に抵触しない限り有効であると宣言されます。


第29条

この大統領令の施行規則は、この大統領令が法律化されてからから30日以内に設定されなければならない。


第30条

この大統領令は、法律化された日付を以って発効する。
